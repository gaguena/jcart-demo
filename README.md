Banco de dados: 
Oracle necessário instalar o JAR no repositorio local do maven.
```shell
mvn install:install-file -Dfile="local\ojdbc8.jar" -DgroupId=com.oracle -DartifactId=ojdbc8 -Dversion=12.2.0.1 -Dpackaging=jar

```
